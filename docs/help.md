# 前置设置

## 游戏设置

* 不论是窗口化还是全屏，游戏都必须是16:9。分辨率最好不低于1280×720

## SAA软件设置

* 核心设置：首次下载必须检查
* 软件启动时检查更新：建议开启
* 展示OCR结果：出现非预期行为可以打开看看是不是识别错别字了，如果存在错别字则去左侧`替换表`上填上新的规则
* 展示运行时窗口截图：建议关闭
* 保存缩放比例：建议关闭
* 代理设置：填写梯子端口，没有则置空
