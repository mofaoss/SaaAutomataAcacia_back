## ✨功能介绍
> [!Tip]
> **Update**
> 1. **隐身模式**上线，游戏将可以完全隐身后台
> 2. 安卡小助手现在完全重构，安装包减少25%，兼容性更好，速度更快，UI更简洁
> 3. 新增**任务自定义列表和计划/定时执行**功能
> 4. 新增繁体中文和英文支持，支持繁中游戏运行
> 5. 新增刷取常规后勤、新星开拓抓帕鲁功能等等新功能
> 6. 增强 OCR 准确率，修复 Steam 登录，体力药剂天数识别；优化记忆嵌片使用；解除窗口化限制

> [!Warning]
>
> 1. 只适配16:9的游戏画面，支持全屏和窗口化，分辨率最好不低于1280×720，越高越好
> 2. 游戏语言必须是简体或繁体中文
